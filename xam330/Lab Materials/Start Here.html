Before you begin

This is a Xamarin.Forms class. To complete the coding exercises, you will need a Windows or Mac development environment with either a simulator, an emulator or a device to run the exercise on. Please see the setup page if you need help installing the Xamarin development environment.

Overview

In this class you will learn how to customize the appearance of Xamarin.Forms visual elements using platform specific APIs by creating and applying effects.

Objectives

  1. Customize control appearance
  2. Apply effects to controls
  3. Create an effect
  4. Add configurable properties

Lecture

XAM330 - Xamarin.Forms Effects

Exercises

  1. Use platform-specific themes
    Exercise
    In this exercise, we will use platform themes to customize our application's appearance.
  2. Add and use an existing Effect
    Exercise
    In this exercise, we will use a provided custom font effect and apply it to a Xamarin.Forms label.
  3. Create a RoutingEffect
    Exercise
    In this exercise, we will create a RoutingEffect and use to apply an effect in XAML.
  4. Create a custom Effect
    Exercise
    In this exercise, we will create a brand new gradient effect and apply it to a button.
  5. Add an attached property
    Exercise
    In this exercise, we will add an attached property to set the second color of our gradient effect.
