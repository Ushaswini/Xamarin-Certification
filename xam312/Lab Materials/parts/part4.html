    2. How do I turn on the Quick Index?

    How do I turn on the Quick Index?

    The ListView control supports a "quick index" drawn just to the right of the control to quickly jump to a grouping sesion. This is activated by setting the GroupShortNameBinding property to a binding that identifies the property on the group that supplies the string to put into the index. This feature is only implemented currently on iOS and Windows.

Steps

Turn on a quick index

One other feature you can use when grouping is turn have a "quick index" jump list which runs on the right side of the ListView. This feature is only available on iOS and Windows, so you will need to run the app on one of those two platforms to see it work.

  1. Set the ListView.GroupShortNameBinding property to data bind to the Key property of your grouped collection.
  2. Run the application to see the quick index.

Show Code

<ListView ... 
           IsGroupingEnabled="true"
           GroupDisplayBinding="{Binding Key}"
           GroupShortNameBinding="{Binding Key}">

  1. Notice that the quick index is cutting off our "favorites" star and the text element with the person's age. Fix this by adding an empty column of "10" units to the Grid in the ViewCell; this will add some padding to the edge of the cell.

Summary

In this exercise, we implemented the quick index feature to improve navigation within a large list of items in a ListView.
