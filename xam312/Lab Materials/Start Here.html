Before you begin

This is a Xamarin.Forms class which includes projects for iOS, Android, and Windows. When working on the exercises, you can select the easiest supported platform to run your application on. Refer to the Setup Requirements on the Xamarin University website if you need help with your development environment.

Overview

This session dives deeper into using and customizing the ListView control in Xamarin.Forms to display collections of data. It is a follow-on class to XAM311 - Using the ListView in Xamarin.Forms and it is expected that you have already taken that class.

Objectives

  1. Create custom cell definitions
  2. Add headers and footers
  3. Separate your data into Groups
  4. Performance-tune your ListViews

Lecture

XAM312 - Customizing the ListView in Xamarin.Forms.

Exercises

  1. Providing a custom cell template for our ListView
    Exercise
    In this first exercise, we will replace an existing ImageCell definition with a custom row visualization that allows us to dictate the position and color of the elements displaying the data.
  2. Add a header and footer to the ListView
    Exercise
    In this exercise, we will add add a header and footer to our ListView.
  3. Adding Grouping support to our Character List
    Exercise
    In this group exercise, we will add grouping support to the ListView so that the characters are grouped by the first letter of the name.
  4. Add a Quick Index
    Exercise
    In this short exercise, you will turn on support for the "quick" jump index for iOS and Windows.
  5. Turn on recycle caching
    Exercise
    In this final exercise, you will turn on the recycle caching strategy.
